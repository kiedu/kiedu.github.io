Системні Вимоги

Для вивчення дисциплін та технологій цього циклу передбачені мінімальні системні вимоги до ваших апаратних ресурів

1. Персональний комп'ютер

Мінімальні вимоги: 2-4 core CPU (1.6 GHz+) / 4GB RAM / 64GB SSD

2. Операційна система

Дистрибутив ОС GNU/Linux на архітектурі x86_64 або аналог ОС Windows.

3. Програмне забезпечення

Встановлений інтерпретатор Python версії не нижче 3.10+. Система контролю версій Git 2.43+. Застосунок для контейнеризації Docker версії 28+.
